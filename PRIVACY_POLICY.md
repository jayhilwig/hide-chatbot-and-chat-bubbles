# Privacy Policy

Last updated: June 26, 2026

Hide Chatbot & Chat Bubbles does not collect, transmit, sell, or share personal information.

## Data Collection

This extension does not collect:
- Personal information
- Browsing history
- Website content
- User activity
- Analytics or usage data

## Local Storage

The extension stores only user preferences using Chrome Storage, including:
- Whether blocking is enabled globally
- Websites where blocking is disabled

These preferences remain on the user's device (or within their Chrome Sync account if they have Chrome Sync enabled) and are never transmitted to the developer.

## Third Parties

This extension does not send data to any external servers and does not share user data with third parties.
